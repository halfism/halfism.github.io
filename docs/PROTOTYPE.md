## 1. 设计理念

### 1.1 设计原则

| 原则 | 描述 | 实现方式 |
|------|------|----------|
| **极简主义** | 去除一切不必要的装饰 | 大量留白，聚焦内容 |
| **专业感** | 展示技术实力与个人品牌 | 暗色主题，现代排版 |
| **可访问性** | 适配各类用户和设备 | 响应式设计，键盘导航 |
| **性能优先** | 秒开页面，无阻塞 | CDN 资源，优化加载 |

### 1.2 参考项目

- **Tailwind CSS 文档** - 简洁的导航 + 内容区域
- **Vercel 官网** - 深色主题 + 渐变高亮
- **Apple 官网** - 大量留白 + 精致动效
- **GitHub Dark** - 开发者友好的配色方案

### 1.3 目标用户

1. 潜在雇主 / 招聘方
2. 技术同行 / 开源社区
3. 潜在客户 / 合作伙伴
4. 技术博主 / 内容创作者

## 5. 响应式断点

| 名称 | 宽度 | 布局调整 |
|------|------|----------|
| mobile | < 640px | 单列，垂直堆叠 |
| tablet | 640-1024px | 双列网格 |
| desktop | > 1024px | 多列网格 |

### 移动端适配

1. **导航**：汉堡菜单，滑出抽屉
2. **英雄区**：垂直堆叠，图片居中
3. **项目卡片**：单列展示
4. **技能墙**：两列网格
5. **页脚**：居中对齐

## 6. 动画规范

### 6.1 全局动画

| 动画名称 | 用途 | 参数 |
|----------|------|------|
| fade-in | 元素进入 | opacity 0→1, 600ms ease |
| slide-up | 元素进入 | translateY 30px→0, 600ms ease |
| scale-in | 元素进入 | scale 0.95→1, 400ms ease |

### 6.2 交互动画

| 交互 | 动画 | 时长 |
|------|------|------|
| 悬停卡片 | 上移 2px + 阴影增强 | 200ms |
| 点击按钮 | 缩小 95% | 100ms |
| 打开菜单 | 滑入 | 300ms |
| 关闭菜单 | 滑出 | 200ms |

### 6.3 特殊动画

| 元素 | 动画类型 | 说明 |
|------|----------|------|
| Hero 头像 | 3D 旋转 | 悬停时归正 |
| 打字效果 | 逐字显示 | 100ms/字 |
| 浮动卡片 | 弹跳 | 3-4s 循环 |
| 技能进度 | 宽度动画 | 加载时触发 |

## 7. 无障碍规范

### 7.1 键盘导航

- [ ] 所有交互元素可通过 Tab 聚焦
- [ ] 聚焦状态有明显的视觉指示
- [ ] 模态框支持 Escape 关闭
- [ ] 导航支持方向键操作

### 7.2 屏幕阅读器

- [ ] 使用语义化 HTML 标签
- [ ] 图片提供 alt 文本
- [ ] 按钮有清晰的标签
- [ ] 链接有描述性文本

### 7.3 色彩对比

| 文本类型 | 最小对比度 | 要求 |
|----------|------------|------|
| 正文 | 4.5:1 | WCAG AA |
| 大字 | 3:1 | WCAG AA |
| 组件边缘 | 3:1 | WCAG AA |

### 8.1 加载性能

| 指标 | 目标值 |

| First Contentful Paint | < 1.5s |
